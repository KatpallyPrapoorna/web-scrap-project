# web-scrap-project
# 🛒 Amazon Price Tracker using Web Scraping

## 📌 Overview
This project is a **web scraping application** that extracts product details from Amazon and tracks price changes over time.

It automatically collects:
- Product title  
- Product price  
- Date of extraction  

The data is stored in a CSV file for further analysis.

---

## 🎯 Objective
- To automate data collection from websites  
- To track product prices over time  
- To perform basic data analysis using collected data  

---

## 🛠️ Technologies Used
- Python  
- BeautifulSoup  
- Requests  
- Pandas  
- CSV  
- Jupyter Notebook / Google Colab  

---

## ⚙️ Features
- 🔍 Scrapes product title and price from Amazon  
- 📅 Stores data with date  
- 📊 Saves data into CSV file  
- 🔄 Supports continuous data collection (automation-ready)  

---

## 🧠 How It Works
1. Sends HTTP request to Amazon product page  
2. Parses HTML using BeautifulSoup  
3. Extracts:
   - Product title  
   - Product price  
4. Stores data in CSV file  
5. Can be automated to run periodically
